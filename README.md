# Quant Sentiment Signals  
*A machine learning and NLP-powered system for predicting financial market movements.*

## 📌 Project Overview  
This project applies **machine learning** and **natural language processing (NLP)** to predict short-term asset price movements. It integrates both **market data** (historical prices, technical indicators) and **sentiment data** (news, social media, market fear & greed index) to generate predictive trading signals.  

The project demonstrates the **end-to-end workflow** of a quantitative research pipeline:  
- Data collection & cleaning  
- Feature engineering (technical + sentiment features)  
- Model development & backtesting  
- Performance evaluation (returns, Sharpe ratio, precision/recall)  

## 🎯 Objectives  
- Showcase expertise in **ML for finance**  
- Build a **production-style pipeline** with reproducibility in mind  
- Evaluate the effectiveness of sentiment + technical signals  
- Provide a solid foundation for future extension into stocks or crypto  

## ⚙️ Tech Stack  
- **Languages**: Python  
- **Data Sources**: Yahoo Finance, Alpha Vantage, Sentiment APIs (Twitter/Reddit, Fear & Greed Index)  
- **Libraries**:  
  - `pandas`, `numpy`, `scikit-learn`, `statsmodels`  
  - `matplotlib`, `seaborn`  
  - `nltk`, `transformers` (for NLP sentiment)  
- **Backtesting**: Custom pipeline + financial metrics
